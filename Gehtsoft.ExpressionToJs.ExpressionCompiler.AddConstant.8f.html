static public method ExpressionCompiler.AddConstant

Brief

Renders a single value as the JavaScript literal the compiler would emit for it.

Declaration
C#
string  AddConstant (object constantValue)

Parameters
constantValue

The value to render. Its runtime type selects the literal form (string, number, boolean, date, enum, ...); null renders as null. A type the built-ins don't recognize throws.

Details

Use it when you are building or testing JavaScript fragments by hand and need one constant formatted the same way the compiler does; it covers only the built-in types and does not apply any custom ExpressionCompiler.Constants registrations, so prefer a full ExpressionCompiler for real translation.
