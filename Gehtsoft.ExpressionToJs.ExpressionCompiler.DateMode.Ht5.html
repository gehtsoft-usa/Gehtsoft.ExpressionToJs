public property ExpressionCompiler.DateMode

Brief

How DateTime values are framed in the emitted JavaScript - local time or UTC.

Declaration
C#
Gehtsoft.ExpressionToJs.DateTimeMode  DateMode public get; public set;

Details

Selects DateTimeMode.Local (default) or DateTimeMode.Utc . Set it before reading ExpressionCompiler.JavaScriptExpression . The host must bind JS-side dates in the same frame (see the library docs / PLAN.md).
