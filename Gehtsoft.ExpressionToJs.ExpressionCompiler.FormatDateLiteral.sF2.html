static public method ExpressionCompiler.FormatDateLiteral

Brief

Renders a fixed date as the JavaScript Date literal the compiler would emit for it.

Declaration
C#
string  FormatDateLiteral (DateTime dt, Gehtsoft.ExpressionToJs.DateTimeMode mode)

Parameters
dt

The date to render. A midnight value emits a date-only constructor; any non-zero time component emits the longer form down to seconds. The value's is ignored - alone decides the frame.
mode

Which frame to build the date in: DateTimeMode.Local for a local-time constructor, DateTimeMode.Utc for a UTC one. This must match how the rest of the page interprets dates or the literal will be off by the client's offset.

Details

Use it when hand-authoring a JavaScript fragment that must agree with compiled expressions on date framing - call it with the same ExpressionCompiler.DateMode you give the compiler so both sides build dates the same way.
