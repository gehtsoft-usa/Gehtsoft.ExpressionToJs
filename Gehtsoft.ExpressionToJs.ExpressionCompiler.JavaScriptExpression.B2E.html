public property ExpressionCompiler.JavaScriptExpression

Brief

The emitted JavaScript - the whole reason to use this type.

Declaration
C#
string  JavaScriptExpression public get;

Details

Read it after the compiler is fully configured (date mode and any custom registrations applied); the result is computed on first access and cached, so later configuration changes are not reflected. Send the string to the client to evaluate, having first loaded the runtime stub ( ExpressionToJsStubAccessor.GetJsIncludesAsString ).
