public property ExpressionCompiler.Methods

Brief

Registry for mapping method calls the built-ins don't cover to JavaScript.

Declaration
C#
Gehtsoft.ExpressionToJs.IJsMethodRegistry  Methods public get;

Details

Register here when the lambda calls a method the built-ins don't handle (a domain helper, an overload with different arguments) so it maps to JavaScript instead of throwing at compile time. Register before reading ExpressionCompiler.JavaScriptExpression ; your registrations are consulted ahead of the built-ins, which also lets you deliberately override one.
