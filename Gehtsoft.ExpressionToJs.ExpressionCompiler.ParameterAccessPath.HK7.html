static public method ExpressionCompiler.ParameterAccessPath

Brief

Builds the dotted member path of a parameter access, for use inside a custom parameter

Declaration
C#
string  ParameterAccessPath (Expression access)

Parameters
access

The member chain rooted in the parameter (as handed to your function).

Details

binding.

Turns a member chain rooted in a rule parameter into a path - m.Address.PostalCode becomes Address.PostalCode, and the bare parameter becomes an empty string. Call it from the parameterAccess function you pass to

IJsParameterRegistry.Map so you don't have to walk the expression yourself.
