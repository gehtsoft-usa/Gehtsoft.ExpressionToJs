public property ExpressionCompiler.Parameters

Brief

Registry for binding a parameter type to a host-side lookup instead of emitting it verbatim.

Declaration
C#
Gehtsoft.ExpressionToJs.IJsParameterRegistry  Parameters public get;

Details

Register a model type here (with IJsParameterRegistry.MapReference ) so rules over that type emit against the form through a reference('path') hook rather than a server-shaped object - the form-validation case, with no subclassing. Register before reading ExpressionCompiler.JavaScriptExpression .
