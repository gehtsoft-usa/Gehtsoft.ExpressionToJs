static public method ExpressionToJsStubAccessor.GetJsIncludesAsString

Brief

Returns the full jsv_* JavaScript runtime stub to embed in your page.

Declaration
C#
string  GetJsIncludesAsString ()

Details

Call it once during page composition and place the returned text inside a script block so the browser has it loaded before any compiled expression runs. The content is fixed and cached after the first call, so repeated calls are cheap and you may call it freely from view code.
