static public method Functions.Fractional

Brief

Gets the part of a number after the decimal point, for whole-number or decimal-place checks.

Declaration
C#
double  Fractional (double value)

Parameters
value

The number to inspect. The sign is ignored: the magnitude is taken first, so -3.25 yields 0.25 exactly as 3.25 does. An integral value yields 0.

Details

Use it to reject values that must be whole (Functions.Fractional(x) == 0) or to validate that a money amount has at most two decimal places.
