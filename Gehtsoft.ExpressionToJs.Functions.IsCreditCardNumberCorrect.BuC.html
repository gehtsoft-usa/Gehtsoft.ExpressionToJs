static public method Functions.IsCreditCardNumberCorrect

Brief

Validates a payment card number with the Luhn checksum, client and server alike.

Declaration
C#
bool  IsCreditCardNumberCorrect (string value)

Parameters
value

The card number as entered. Embedded spaces and dashes are tolerated (so grouped input like "4111 1111 1111 1111" validates); null or any other non-digit character makes the result false.

Details

Use it to catch typos in a typed-in card number before submission. It confirms the number passes the Luhn check only; it does not verify the card exists or is active.
