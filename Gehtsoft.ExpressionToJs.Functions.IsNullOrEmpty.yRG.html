static public method Functions.IsNullOrEmpty

Brief

Fluent "required text" check - true when the string is missing or empty.

Declaration
C#
bool  IsNullOrEmpty (this string v)

Parameters
v

The string to test. Both null and "" yield true; whitespace-only text does not.

Details

Use it where a form treats both a missing value and an empty string as "not provided", which is usually what you want.
