static public method Functions.ToBool

Brief

Coerces a loosely-typed field to a bool, accepting the common yes/no spellings.

Declaration
C#
bool  ToBool (object v)

Parameters
v

The value to coerce. null becomes false; an actual passes through; a string is matched case-insensitively - "true"/"yes"/"1"/"on" give true and "false"/"no"/"0"/"off" give false. Any other string, or any other type, is rejected with an exception, so use it only where the input is known to be one of these shapes.

Details

Use it to give a checkbox, text input, or query-string value definite boolean meaning inside a rule, rather than matching only the literal "true"/"false".
