static public method Functions.YearsSince

Brief

Counts completed calendar years between two dates - the usual way to check age.

Declaration
C#
double  YearsSince (DateTime args0, DateTime args1)

Parameters
args0

The reference (usually later) date, e.g. today; its time component is discarded.
args1

The baseline (usually earlier) date, e.g. a birth date; its time component is discarded. The argument order sets the sign of the result.

Details

Use it for the canonical "must be at least N years old" rule (Functions.YearsSince(DateTime.Today, birthDate) >= 18): the count advances only on a completed calendar year, so it matches how age is reckoned in practice.
