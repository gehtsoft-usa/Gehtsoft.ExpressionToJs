interface IExpressionEmitContext

Brief

The building blocks a method-call translator needs to emit JavaScript.

Details

Implemented by ExpressionCompiler ; delegates to its (overridable) walk/escape helpers so custom translators compose with subclass customizations such as parameter rendering.

Public Properties

DateMode

The configured DateTime framing (Local or Utc).

Public Methods

Emit

Emit the JavaScript for a sub-expression (walks the tree).

EmitParameterAccess

Emit access to a parameter-rooted member/indexer chain.

EmitPredicate

Emit a lambda as a JavaScript function-expression predicate (function (...) { return ...; }).

EscapeRegex

Escape a regex pattern for use inside a JavaScript /.../ literal.

EscapeString

Render a CLR string as a JavaScript string literal.

RootsInParameter

True if the expression is (or roots in) one of the lambda's parameters.

TryEvaluate

Try to fold a (parameter-free) sub-expression to a constant value.
