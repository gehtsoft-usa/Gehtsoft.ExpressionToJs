interface IJsMethodRegistry

Brief

Additive extension surface for mapping method calls to JavaScript.

Details

Registrations are consulted before the (fixed, correctly-ordered) built-ins, so a consumer can shadow a specific method without being able to reorder or remove built-ins - the built-in precedence invariants are preserved. Registrations are tried in the order added.

Public Methods

AddTranslator

Register a custom translator (consulted before built-ins).

MapMethod

Map a method (by type + name) to a JavaScript template ($obj, $0, $1, ...).

MapMethod

Map a specific method to a JavaScript template ($obj, $0, $1, ...).
