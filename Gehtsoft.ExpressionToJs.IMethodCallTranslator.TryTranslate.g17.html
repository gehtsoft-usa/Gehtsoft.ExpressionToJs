public method IMethodCallTranslator.TryTranslate

Brief

Implement this to claim a method call and produce its JavaScript.

Declaration
C#
bool  TryTranslate (MethodCallExpression call, Gehtsoft.ExpressionToJs.IExpressionEmitContext context, out string js)

Parameters
call

The call to inspect; check its method, declaring type, and arguments to decide whether it is yours.
context

Use it to emit sub-expressions, fold constants, escape strings, and read the date mode - so your output composes with the rest of the compiler.
js

Receives the emitted JavaScript when you return true; otherwise leave it null.

Details

The compiler offers each call to the registered translators in turn, so return false (and leave null) for calls you don't handle, and only consume the ones you recognize - keeping translators disjoint is what lets them be combined freely.
